# For Array/List:

O(1) means constant time

Operation                       Worst case
Copy                            O(n)
Append                          O(1) 
Pop last                        O(1)
Pop intermediate                O(n)
Insert                          O(n)
Get item                        O(1)
Set item                        O(1)
Delete item                     O(n)
Iteration                       O(n)
Get slice                       O(k)
delete slice                    O(n)
Set slice                       O(k+n)
Extend                          O(k)
Sort                            O(nlogn)
Multiply                        O(nk)
Length                          O(1)
